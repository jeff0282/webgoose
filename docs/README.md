# Webgoose Documentation

Welcome to the Webgoose Documentation!

These documents aim to introduce all of the concepts, behaviours, and tools used by Webgoose.

Below is a description of the contents of all the documents here.


## Table Of Contents:


### Introductory Documents:

**[Getting Started](getting_started.md)**
An introduction to the Webgoose Static Site Generator, alongside instructions and guidance on how to set up a Webgoose site.


**[Install & Usage](install_and_usage.md)**
Provides Information on how to install Webgoose, as well as documenting all of the commands and arguments supported.


**[Authoring Pages](authoring_pages.md)**
Information and guidance on how pages are written in Webgoose and some tips and tricks.


**[File Handling](file_handling.md)**
Documents how Webgoose handles files, including: Pages vs Static Files, the final structure of the built site, and other things.


**[Templates](templates.md)**
Introduces the Jinja2 Templating Engine, and how Webgoose uses it to generate pages and make maintaining a website easier.


**[Previewing Your Site](previewing_your_site.md)**
How to use Webgoose to preview your site locally before putting your site on a web server.


### The Cool Stuff:

**[Template Variables](template_variables.md)**
Defines the variables and methods available in Page and Template Files.


**[Template Filters & Functions](template_filters_and_functions.md)**
Defines the additional filters and functions available in Page and Template Files to manipulate and process data.


**[The Config File](the_config_file.md)**
Defines all of the necessary options available in the config file, as well as additional options and default values.


**[The Abstract Directory Tree](abstract_directory_tree.md)**
Documentation for the Abstract Directory Tree, the API it provides, and how to use it to interact with your site.


**[Data Files](data_files.md)**
Provides information on how the load data files for use in Page and Template files.
